# Modular Ability System - Unity 3D

### 🧠 Overview

A modular and extensible ability system for Unity designed to simplify the creation of gameplay abilities using reusable components and data-driven configuration.

The architecture prioritizes scalability, maintainability, and clean separation of runtime logic from configuration data.

---

### 🚧 Project Status

This project is currently in active development. New features and improvements are being added continuously.

### ⚡ Current focus
- Permanent and conditional status effects
- Status stacking system

---

### 🧱 Core Design Principles

- Data-driven design (ScriptableObject-based configuration)
- Event-driven communication system
- Modular ability architecture (composable components)
- Clear separation of configuration and runtime state
- High extensibility for future gameplay systems

---

### ⚙️ Features

- Configurable ability parameters  
- Cooldown system  
- Cast time & cast progress tracking  
- Interruptible casting mechanics  
- Damage over Time (DoT) effects  
- Customizable duration and damage scaling  
- Event bus for decoupled communication  

---

### 🌱 Planned Features

- Resource system (e.g. mana, energy, stamina)  
- Buff & debuff framework  
- Area of Effect (AoE) abilities  
- Ability combinations / chaining system  
- Channeling system  
- Cooldown reduction mechanics  
- Damage type & resistance system  
- Demo scene showcasing full system
